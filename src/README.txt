SysMon — монитор системы для macOS (menu bar)
==============================================

Установка на новый Mac:
  1. Распакуйте архив SysMon-installer.zip
  2. Откройте Terminal в распакованной папке
  3. Запустите:  bash install.sh
     (или двойной клик по install.sh после chmod +x)

Что делает установщик:
  • копирует SysMon.app в /Applications
  • снимает атрибут карантина (Gatekeeper не будет блокировать)
  • переподписывает приложение ad-hoc подписью на этой машине
  • предлагает включить автозапуск при входе (LaunchAgent)
  • запускает приложение — иконка появится в строке меню

Если macOS всё равно блокирует:
  Системные настройки → Конфиденциальность и безопасность
  → внизу будет «SysMon заблокирован» → Открыть всё равно.

Требования:
  • macOS 12 (Monterey) или новее
  • Apple Silicon или Intel (универсальный бинарник)

Управление:
  • Клик по иконке в строке меню — текущие метрики
  • «Открыть дашборд…» (⌘D) — окно с графиками
  • «Запускать при входе» — переключатель автозапуска
  • «Выйти» (⌘Q)

Удаление:
  • Перетащить /Applications/SysMon.app в Корзину
  • Удалить ~/Library/LaunchAgents/com.sysmon.menubar.plist
    (если включали автозапуск)
